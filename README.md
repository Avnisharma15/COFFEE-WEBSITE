# Coffee Website

This is a simple and responsive coffee shop website made using **HTML, CSS, and JavaScript**.  
It includes different sections like Home, About, Menu, Testimonials, Gallery, and Contact.

## Features
- Clean and modern design  
- Responsive layout  
- Image gallery  
- Testimonials slider  
- Contact form UI  
- Easy to understand code  

## Technologies Used
- HTML  
- CSS  
- JavaScript  
- Font Awesome Icons  
- Swiper.js (for slider)

## How to Use
1. Download or clone the project  
2. Open `index.html` in your browser  
3. All images are inside the `Coffee/images` folder  

## Folder Structure
coffee-website/
│── index.html
│── style.css
│── script.js
│── Coffee/
    └── images/
